# 核心成员须知

2022.01.19.1.1 - 国际基岩服

▶欢迎任何热爱MC的玩家，本服旨在建设一个友好的原版生存社区。

▶保持真诚的态度，不作弊、不刷物品、不违反禁止条例。

▶爱国、敬业、诚信、友善 。

# 一、简介

欲申请核心成员，需熟读了解本须知

# 二、义务与责任

  • 服务器白名单审核与添加

  • 调解服务器日常矛盾，维护服务器和平稳定

  • 指引新人，引导服内成员进行生存建筑，引导玩家交流环境

  • 礼貌待人，友好相处、团结友爱、遵纪守法

    注：若不满足核心成员基本要求，则会撤除核心权限。

# 三、申请

# 申请条件

  • 或在服务器建过很多建筑，有很多成就。
  • 或有一定的想法和技术（策划、宣传等）
  • 服内游玩时间比较长
  • 踏实认真，待人友善，尊重他人
  • 能冷静分析和处理服务器的各种事件

# 申请方式

  • 向管理员告知申请，评估是否满足申请条件，三天只能申请一次
  • 阅读群文件--国际基岩服--《核心成员须知》
  • 填写问卷，提交后通知管理员
  • 管理员评估问卷结果并在游戏中授予核心成员权限

# 四、权限一览

  • 新人审核 | 添加服务器白名单

    新人必须加入QQ群，离开QQ群将移除白名单。

    新人需填写群文件——在线文档——国际服白名单申请 的收集表

    收集表填写结果经服主审核后，核心成员才可添加新人到服务器白名单。

    /white add <玩家准确的昵称> -- 添加服务器白名单

    /white list -- 列出白名单

    注：联系管理员删除白名单 : /white remove

  • 开启世界规则（生物破坏、TNT爆炸、火焰蔓延）

  • 记录公共坐标，删除任意公共坐标

    信任成员可以记录公共坐标，但只能删除自己创建的公共坐标

    核心成员可以删除任意的公共坐标
