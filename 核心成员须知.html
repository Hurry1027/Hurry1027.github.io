# 核心成员须知

2023.06.29 - 国际基岩服

▶欢迎任何热爱MC的玩家，本服旨在建设一个友好的原版生存社区。

▶保持真诚的态度，不作弊、不刷物品、不违反禁止条例。

▶爱国、敬业、诚信、友善 。

# 一、简介

欲申请核心成员，需熟读了解本须知

# 二、义务与责任

  • 调解服务器日常矛盾，维护服务器和平稳定

  • 指引新人，引导服内成员进行生存建筑，引导玩家交流环境

  • 礼貌待人，友好相处、团结友爱、遵纪守法

    注：若不满足核心成员基本要求，则会撤除核心权限。

# 三、申请

# 申请条件

  • 核心成员将具有一定的管理权限， 所以并不是信任成员的简单升级。
  • 踏实认真，待人友善，尊重他人
  • 能冷静分析和处理服务器的各种事件
  • 或在服务器建过很多建筑，有很多成就。
  • 或有一定的想法和技术（策划、宣传等）
  • 服内游玩时间比较长

# 申请方式

  • 向管理员告知申请，评估是否满足申请条件，三天只能申请一次
  • 阅读群文件--国际基岩服--《核心成员须知》
  • 填写问卷，提交后通知管理员
  • 管理员评估问卷结果并在游戏中授予核心成员权限

# 四、权限一览

  • 禁言：/mute true /mute false 禁言开启后只允许核心成员发言。
