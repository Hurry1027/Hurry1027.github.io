# 二、Rxree1事件

4月20日，僵尸猪与Rxree1发生冲突。因僵尸猪未经同意在Rxree1领地附近设置传送点，Rxree1因其不尊重私人领地而不满。事后Rxree1直接提出，僵尸猪不但不能意识到自己的问题，继续与Rxree1激烈对峙，Rxree1因此退出服务器。

# 三、prc退镇风波

6月18日。此时群内已有很多人对僵尸猪的言行作风感到不满。由于门主不会建筑将新亭镇过户给僵尸猪，prc nicozhu对新亭镇日后发展产生忧虑，所以选择退出新亭。而僵尸猪面对prc的退出，百般阻挠，直言“不能忘本”、“退出条件是给每人一个合金块”等霸道无理的要求，随后creeperyun提出也要退出新亭镇，僵尸猪反应失态，以各种不当理由疯狂诋毁prc。后由服主进行事件总结，驳回了僵尸猪的无理要求，prc顺利退出新亭镇。

6月19日，prc发现丢失一个装有大量珍贵物品的潜影盒丢失，经查，确认是僵尸猪在搬迁prc住宅时所偷取，并且将其放入了自己家的箱子内。除此之外，后又发现僵尸猪偷取了51个金苹果、一整套合金装备。

面对服主与prc的质问，当时僵尸猪由于拍摄毕业照未能及时回复

当天晚上僵尸猪面对事实证据，未能成功解释其持有prc遗失物品的来历，且脾气暴躁，不知所云，最后僵尸猪承认拿取潜影盒初心不正，承认在之前Againchi搬迁时通过非法手段刷物品，这已触碰服务器禁止条例。面对质疑，僵尸猪非但不配合事实调查，仍言辞激烈，愤怒退群。

由于事实证据确凿，最终判定其非法盗取prc重要财产若干，违反服务器禁止以不正当方式刷物品的条例。其本人多次出言不逊，语言暴力他人，引起服内众人强烈不满，影响恶劣。

采取封禁措施：无期封禁。

# 四、总结

由于每个人成长经历不同，性格不同，有人性格难免急躁易怒，服务器给予一定程度的理解，并及时劝解，希望其能与其他成员互相尊重，友好交流。然而僵尸猪多次在违规边缘试探，其言行引发众人强烈不满，在其他人多次指责并提出意见后，仍然不去思考自身问题，我行我素，延续其一贯作风，这不符合服务器的核心价值观。

其次，除暴躁性格外，僵尸猪在多个事件中表现出的，对于其他成员、他人财产、领地的不尊重，在各个事件中在底线上不择手段擢取最大利益的本性，对个人错误拒不道歉及改正，作风失德失态，造成了极大的公共负面影响，遭到多数成员的一致唾弃。

服务器能够理解人与人之间成长的不平衡性与不同的心理特质，会尽量包容敢于承认问题，积极改正错误的玩家。而对于屡教不改，损害、不尊重他人权利的成员，将不再给予机会，直接踢出。

# 五、评价

（一）Wn1027评价

（主观性评价，会带有一定偏见，仅供参考讨论）

僵尸猪的性格特质。僵尸猪的行为与普通的熊孩子不一样，熊孩子只是认知不清，是非善恶分辨尚模糊，所以做出违规的事情。而僵尸猪的边缘行为则很多事来源于他的性格特质。

1、性格暴躁易怒，不喜欢别人的否定，以激烈方式来回应外界的否定，以此来保护自身，这样的情绪外化方式十分扭曲。

其内心并不是强大的，在群内的各种自曝弱小的行为（做手术、抑郁症、各种过往私事），本质上是为了争取同情和理解，希望他人能理解他的弱小内心，但由于其言辞激烈，外界对此的一般性认识是故意卖惨博同情。又根据其在群内日常言辞轻浮、自曝炫耀照片、半夜打电话给我的行为，也可见其希望能获取外界认同感，可见其内心长期缺乏安全感，稳重感。

心理学中其实有类似这样的分析，容易情绪化脾气暴躁的人，其实是在维护自己的一些东西。

但是站在他自己的角度，日常言行中的表达不当和过激会造成外界对他的反感与厌恶，这更加剧了外界对他的否定，甚至造成与他人之间的直接冲突。在此期间，已经有许多人直接指明其言行失态的地方。但由于其不会处理矛盾，或是出于自尊心，或是没有思辨能力认识不到自己错误，拒绝道歉与改正，没有公共道德自觉。导致其在众人心中形象变差，隔阂加深直至破裂。

在这个过程中，不仅对他个人无益，同时更是造成了对他人的语言暴力，伤害了他人。而在事后，其本人不会或拒绝改正错误，以这样扭曲的方式与人相处，自然会招致众人嫌弃，影响公共秩序与氛围，此时服务器没有理由再留下他。

个人特殊情感勿要侵入公共生活领域。虽然性格原因会导致言行失态，这本是自然的，但是作为一个成年人，应当负担起一定的社会责任，应当学会充分尊重事实，尊重他人，控制个人不当言行，控制住不当的情绪外化，情绪的释放和表达应当有正确的方式。面对事件，情绪可以帮你辨别是非好坏，是人基本的道德天性，但在实际行动中应当以理性的态度去看待和做事。但是性格特质绝不能作为伤害他人，暴力他人的辩护理由。

知错能改是有必要的。每个人在成长过程中或多或少都会有错误，但知错能改善莫大焉，一个能不断地自我批判，意识到自己问题并加以改正的人，是最值得肯定的。我们也需要学会理解和喜欢这样一个处在成长变化之中的人。

服务器可以尽可能包容不同年龄阶段，不同心理特质的成员，但是对于给服务器成员造成恶劣影响或破坏的玩家，服务器将不会保留。
