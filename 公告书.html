▶保持真诚的态度，不作弊、不刷物品、不违反禁止条例。

▶爱国、敬业、诚信、友善 。

# 二、生存规则

一般性禁止条例，视具体情况处罚，认错态度好可提前出狱。

  • 禁止未经许可私闯民宅

  • 禁止偷窃不属于自己的物品（标注公共物品除外）

  • 禁止利用游戏不平衡恶性bug刷物品

  • 禁止矿物透视

  • 禁止开挂作弊

  • 禁止故意暴力他人

  • 禁止寻衅滋事

  • 世界规则：

    • 死亡不掉落。

    • 生物破坏、tnt爆炸、火焰蔓延信任成员可开关

    • 死亡惩罚：随机丢失50~100点经验

  • 边界规则：

    • 主世界、下届、末地限制坐标在5120内，资源服不设限。

  • 红石特性机器规则：

    • 不占用末地门只能刷掉落方块的刷沙机可用（不建议使用）
    • 0tick海带机甘蔗机可用（易卡服，不建议使用）
    • 区块羊毛机可用（不建议使用）
    • 火焰蔓延猪人塔不建议使用（需要侦测器高频来刷新方块状态，会卡服）

  • 末地探索规则

    • 新人可以传送至资源世界拿取鞘翅。
    • 每人只可以探索一座带船的末地城，
    • 请在发现末地城后及时记录坐标，拿取鞘翅。
    • 之后你无法靠近未探索的末地城。
    • 鞘翅遗失后联系服主找回。

# 三、服务器管理

# （一）存档管理

  • 服务器每晚22:00备份一次存档，备份存档最多保留七天。

  • 卫星图两小时渲染更新一次

  • 主存档建筑损毁，可从旧存档恢复。（但是恢复很耗时耗力）

  • 玩家聊天、行为记录均可查。

# （二）服主事务

  • 基本服务器维护（维护服务器正常运行，插件BUG反馈）

  • 白名单申请

  • 成员违规追查及处理

  • 信任成员、核心成员申请

  • 店铺许可证申请

  • 出生点群岛建筑申请

  • 服务器音乐推荐（参考下文详细说明）

  • 建筑迁移（8颗钻石，如有地下部分12颗钻石）

  • 鞘翅遗失找回

# 四、服务器成员权限等级

# （一）权限分级

  • 访客：未获得服务器白名单的陌生玩家，无法与世界交互。
  • 萌新：获得服务器白名单的新人，无权使用很多功能。
  • 信任成员：服务器正式成员，受服务器玩家广泛信任，可以控制世界规则，可开启飞行。
  • 核心成员：个人综合素质高，有一定管理权限，可以控制世界规则。
  • 管理员：服务器管理员，可进入服务器后台。
  • 白名单暗号：不偷不熊，否则天打雷劈。

# （二）信任成员申请

  • 不偷不熊，道德素质高
  • 有自己的房子（不是火柴盒）（普遍的申请依据）
  • 能被服内在线的大多数玩家所信任（需要一定的在线时间）
  • 如果你想成为信任成员，可直接向服主提出申请
  • 处理申请时，服主会参考其他玩家意见进行评估，给出答复

# （三）核心成员申请

  • 核心成员将具有一定的管理权限， 所以并不是信任成员的简单升级。
  • 踏实认真，待人友善，尊重他人
  • 能冷静分析和处理服务器的各种事件
  • 或在服务器建过很多建筑，有很多成就。
  • 或有一定的想法和技术（策划、宣传等）
  • 服内游玩时间比较长
  • 具体申请流程参阅《核心成员须知》

# 五、插件功能及游戏操作：

# （一）插件功能

  1. 玩家权限：游客-信任成员-核心成员-管理员

  2. 基础功能服务插件：服主自己写的js脚本（LiteLoaderBDS加载器）

  3. 传送（传送地标，两个个人传送点）

  4. 飞行模式（仅用于建筑，禁止跑图，末地禁用）

  5. 建筑导出（如果你想暂退服务器，可以联系服主用结构方块导出你的个人建筑）

  6. 官方建材商店（物价与村民类似）

  7. 玩家店铺（需建好店铺向服主申请经营许可证，出售或求购，自行设置）

  8. 资源同步服务器（采集资源到资源服，资源服不定期重置存档，目前资源服物理服务器由@wosihaoge提供）

  10. 给村民命名“无敌”可使村民无敌。

# （四）店铺操作指南：

# 1、经营许可证申请

  1. 建好一个店铺（商业街或其他地方）
  2. 向服主申请经营许可证

# 2、添加店铺、添加柜台

  1. 鹦鹉螺壳点击地面，新建店铺
  2. 鹦鹉螺壳点击木桶，新建柜台
  3. 鹦鹉螺壳点击另一木桶，设置该柜台对应的仓库箱
  4. 鹦鹉螺壳点击另一木桶，设置该柜台对应的收款箱
  5. 鹦鹉螺壳点击柜台木桶，设置每份商品的数量，价格物品及价格数量
  6. 1种商品对应1个柜台和1个仓库 仓库内物品必须完全相同，包括特殊值、物品NBT（方块朝向也会影响）
  7. 多个柜台可以设置同一个收款箱
  8. 仓库与收款箱需用木棍锁箱防盗，或者设置个人领地。

# 3、额外柜台配额申请

  1. 默认限制每个玩家最多拥有10个柜台。
  2. 根据经营状况、商店面积、店主是否认真负责，可额外申请20个柜台配额。
  3. 长期经营不善的店铺可能会被强制删除柜台。

# （五）服务器音乐：

# 1、说明：

  1. 使用命令/mucmenu，或使用唱片点击地面，唤出菜单进行播放。
  2. 游戏内下载的资源包在游戏设置-存储-缓存数据内，个人自行定期清理。

# （六）生物捕获：

# 1、说明：

  1. 使用命令/cm true开启生物捕获功能。
  2. 使用<蜜脾>查看已捕获的生物及使用说明。
  3. 每人只允许同时拥有7个生物蛋。
  4. 生物可以通过生物蛋或表单释放，用表单释放后生物蛋将失效。
  5. 捕获者拥有生物所有权，无法转让、售卖。
  6. 可向服主申请将无法捕获的生物添加至可捕获列表。

# 六、加入我们

我们希望建立一个和谐友爱的社区。

我们希望这里不仅是游戏，也是我们的一处栖息地， 超越现实的桎梏，来亲手打造你心中向往的风景。

是亭台楼阁，是温馨小镇，是朋克都市，或是浮空岛屿。 每一处风景，都如梦幻中的秘境。 即使是火柴盒，也能众木成林，为一处奇观。

一无所有的我们，从这里开始， 愿每个人都能从MC中的某个细节之处发现自己所热爱的东西， 或建筑、或红石、或绘画、或代码， 我们都是成长中的玩家，我们在探寻与这个世界交互的方式， 所以，这里不仅仅是游戏。

摒弃互联网的信息轰炸，让浮躁的心灵稍息。 在你的脑海深处，一定有数不尽的天马行空，在这里去实现和分享。

拉进网络的距离，在这里成为朋友、邻居。 少些戾气，多些合作，冷静处事，睦邻友好。

Minecraft的世界是空荡且孤独的。 “成长”究竟是什么呢？ 我们的灵魂深处不断的发出追问，你在世界上的这一生，到底要做些什么？ 学会独立，需要认清以及承受来到世界上的孤独感，去细细感受、认识、学习这个世界。 当真正能够回答内心深处的追问的时候，我们才会找到生命的归宿。

如果你对我们的理念深以为然，欢迎你的到来！ 为了实现这样的梦想，我们需要志同道合的伙伴：

我们需要有独特想法的玩家来一起策划和完善游戏内容。 我们需要会些文笔或视频的玩家来帮助宣传。 我们需要想学编程的玩家来一起用插件实现想法。

尽管现在一无所有， 我们来一起学习、讨论、设计属于自己的服务器， 享受成长的过程，这就是最棒的游戏体验。

可大致分三部分内容：

  1. 宣传：文案、视频、论坛贴吧网站招新。
  2. 策划：理念讨论、游戏机制、玩法设计、活动策划、城市设计规划。
  3. 技术：建筑学习、红石科技、命令方块、插件编程。
