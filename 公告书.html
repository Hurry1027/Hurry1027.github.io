# 二、生存规则

  • 一般各服都有的禁止条例：

    • 禁止未经许可私闯民宅
    • 禁止偷窃不属于自己的物品（标注公共物品除外）
    • 禁止利用游戏不平衡恶性bug刷物品

    • 禁止开挂作弊
    • 禁止故意暴力他人
    • 禁止寻衅滋事

  • 世界规则

    • 死亡不掉落。
    • 死亡惩罚：随机丢失50~100点经验
    • 生物破坏、tnt爆炸、火焰蔓延信任成员可开关

    • 每人只可以探索一座带船的末地城，

    • 已获得鞘翅的玩家无法靠近未探索的末地城。

# 三、服务器管理

# （一）存档管理

  • 服务器每晚22:00备份一次存档，备份存档最多保留七天。

  • 卫星图两小时渲染更新一次

  • 每十五分钟背包备份

  • 玩家聊天、行为记录均可查。

# （二）服主事务

  • 基本服务器维护（维护服务器正常运行，插件BUG反馈）

  • 信任成员、核心成员申请

  • 店铺许可证申请

  • 公共区域建筑申请

  • 服务器音乐推荐（参考下文详细说明）

  • 建筑迁移（8颗钻石，如有地下部分12颗钻石）

  • 个人建筑导出.mcstructure结构方块格式

# 四、服务器成员权限等级

# （一）权限分级

  • 访客：未获得白名单的陌生玩家，无法与世界交互。
  • 萌新：刚刚获得白名单的新人，无权使用很多功能。
  • 信任成员：正式成员，受其他玩家广泛信任，可以控制世界规则，可开启飞行。
  • 核心成员：个人综合素质高，有一定管理权限，可以控制世界规则。
  • 管理员：服务器管理员，可进入服务器后台。
  • 白名单暗号：不偷不熊，否则天打雷劈。

# （二）信任成员申请

  • 不偷不熊，道德素质高
  • 有自己的房子（不是简单火柴盒）（普遍的申请依据）
  • 能被服内在线的大多数玩家所信任（需要一定的在线时间）
  • 如果你想成为信任成员，可直接向服主提出申请
  • 处理申请时，服主会参考其他玩家意见进行评估，给出答复

# （三）核心成员申请

  • 核心成员将具有一定的管理权限， 所以并不是信任成员的简单升级。
  • 踏实认真，待人友善，尊重他人，有理性思考能力
  • 或在服务器建过很多建筑，有很多实体成就。
  • 或有一定的想法和技术（策划、宣传等）
  • 服内游玩时间比较长
  • 具体申请流程参阅《核心成员须知》

# （一）插件功能

# （四）店铺操作指南：

# 1、经营许可证申请

  1. 建好一个店铺（商业街或其他地方）
  2. 向服主申请经营许可证

# 2、添加店铺、添加柜台

  1. 鹦鹉螺壳点击地面，新建店铺
  2. 鹦鹉螺壳点击木桶，新建柜台
  3. 鹦鹉螺壳点击另一木桶，设置该柜台对应的仓库箱
  4. 鹦鹉螺壳点击另一木桶，设置该柜台对应的收款箱
  5. 鹦鹉螺壳点击柜台木桶，设置每份商品的数量，价格物品及价格数量
  6. 1种商品对应1个柜台和1个仓库 仓库内物品必须完全相同，包括特殊值、物品NBT（方块朝向也会影响）
  7. 多个柜台可以设置同一个收款箱
  8. 仓库与收款箱需用木棍锁箱防盗，或者设置个人领地。

# 3、额外柜台配额申请

  1. 默认限制每个玩家最多拥有10个柜台。
  2. 根据经营状况、商店面积、店主是否认真负责，可额外申请20个柜台配额。
  3. 长期经营不善的店铺可能会被强制删除柜台。

# （五）服务器音乐：

# 1、说明：

  1. 使用命令/mucmenu，或使用唱片点击地面，唤出菜单进行播放。
  2. 游戏内下载的资源包在游戏设置-存储-缓存数据内，个人自行定期清理。

# （六）生物捕获：

# 1、说明：

  1. 使用命令/cm true开启生物捕获功能。
  2. 使用<蜜脾>查看已捕获的生物及使用说明。
  3. 每人只允许同时拥有7个生物蛋。
  4. 生物可以通过生物蛋或表单释放，用表单释放后生物蛋将失效。
  5. 捕获者拥有生物所有权，无法转让、售卖。
  6. 可向服主申请将无法捕获的生物添加至可捕获列表。

# 六、加入我们

我们希望建立一个和谐友爱的社区。

我们希望这里不仅是游戏，也是我们的一处栖息地， 超越现实的桎梏，来亲手打造你心中向往的风景。
