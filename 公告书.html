# 会飞的史蒂夫

2023.01.12 - 国际基岩服

《核心成员须知》中列出了与核心成员有关的部分。

▶欢迎任何热爱MC的玩家，本服旨在建设一个友好的原版生存社区。

▶保持真诚的态度，不作弊、不刷物品、不违反禁止条例。

▶爱国、敬业、诚信、友善 。

IP地址：【见群公告】

世界规则：死亡不掉落。关闭生物破坏、TNT爆炸、火焰蔓延（核心成员可开启）

# 二、禁止条例

一般性禁止条例，视具体情况处罚，认错态度好可提前出狱。

  • 禁止未经许可私闯民宅
  • 禁止偷窃不属于自己的物品（标注公共物品除外）
  • 禁止利用游戏不平衡恶性bug刷物品
  • 禁止开挂作弊
  • 禁止故意暴力他人
  • 禁止寻衅滋事

# 三、服务器成员权限等级

# （一）权限分级

  • 游客：无权使用很多功能。
  • 信任成员：正式成员，受服务器玩家广泛信任，可以控制世界规则。
  • 核心成员：玩家素质高，有一定管理权限，可以控制世界规则。
  • 管理员：可以修改玩家档案信息。
  • 白名单暗号：不偷不熊，否则天打雷劈。

# （二）信任成员申请

  • 不偷不熊，道德素质高
  • 有自己的房子（不是火柴盒）（普遍的申请依据）
  • 能被服内在线的大多数玩家所信任
  • 如果你想成为信任成员，可直接向管理员提出申请
  • 处理申请时，管理员会参考其他玩家意见进行评估，给出答复

# （三）核心成员申请

  • 或在服务器建过很多建筑，有很多成就。
  • 或有一定的想法和技术（策划、宣传等）
  • 服内游玩时间比较长
  • 踏实认真，待人友善，尊重他人
  • 能冷静分析和处理服务器的各种事件
  • 具体申请流程参阅《核心成员须知》

# 四、插件功能及游戏操作：

# （一）插件功能

  1. 玩家权限：游客-信任成员-核心成员-管理员

  2. 基础功能服务插件：服主自己写的js脚本（LiteLoaderBDS加载器）

  3. 锁箱子：LockYourChest by KING

  4. 领地：ILAND by Redbeanw

  5. 音乐：可以推荐1-2首音乐加入服务器BGM。LF_Music by FlyNight

  6. 个人&公共坐标记录与广播

  7. 矿物统计、死亡统计、时间统计等

  8. 飞行模式（仅用于建筑，禁止跑图，末地禁用）

  9. 建筑导出（如果你想暂退服务器，可以联系服主用结构方块导出你的个人建筑）

  10. 玩家店铺（需建好店铺向服主申请经营许可证，出售或求购，自行设置）

  11. 资源同步服务器（采集资源到资源服，资源服不定期重置存档，目前资源服物理服务器由@wosihaoge提供）

  12. TrapDoor (opens new window)（下载群文件的TrapDoor资源包，可查看tps、mspt、村庄范围、史莱姆区块等）

  13. 给村民命名“无敌”可使村民无敌。

  6. 播放BGM: /mucmenu 或 使用任意唱片点地板

  7. 店铺菜单: 鹦鹉螺壳点地或木桶（需申请店铺许可证）

  8. 插件更新日志: /update

  9. 领地管理: 主菜单选择或/land

  10. 领地插件指南: iLand 文档站 (opens new window)

  11. TrapDoor: TrapDoor插件指南 (opens new window)

  13. 查看卡顿情况: /prof

  14. 获取地图画: /getmap xxx.jpg

  15. 裁剪地图画: /getmap xxx.jpg cut

# （四）店铺操作指南：

# 1、经营许可证申请

  1. 建好一个店铺（商业街或其他地方）
  2. 向服主申请经营许可证

# 2、添加店铺、添加柜台

  1. 鹦鹉螺壳点击地面，新建店铺
  2. 鹦鹉螺壳点击木桶，新建柜台
  3. 鹦鹉螺壳点击另一木桶，设置该柜台对应的仓库箱
  4. 鹦鹉螺壳点击另一木桶，设置该柜台对应的收款箱
  5. 鹦鹉螺壳点击柜台木桶，设置每份商品的数量，价格物品及价格数量
  6. 1种商品对应1个柜台和1个仓库 仓库内物品必须完全相同，包括特殊值、物品NBT（方块朝向也会影响）
  7. 多个柜台可以设置同一个收款箱

# （五）服务器音乐：

# 1、说明：

  1. 使用命令/mucmenu，或使用唱片点击地面，唤出菜单进行播放。
  2. 每人限推两首，希望它是你单曲循环的想要分享的好音乐，而不是一时兴起的梗曲。
  3. 每两周周末更新一次，必要时请提醒服主更新
  4. 服务器音乐资源包限制30首音乐，会随机删除旧的音乐，除非重新填表。
  5. 游戏内下载的资源包在游戏设置-存储-缓存数据内，个人自行定期清理。
  6. 推荐你喜欢的音乐吧~ (opens new window)

# 五、加入我们

我们希望建立一个和谐友爱的社区。 我们希望这里不仅是游戏，也是我们的生活，是我们的元宇宙。 会飞的史蒂夫，超越现实的桎梏，来亲手打造你心中向往的风景。

是亭台楼阁，是温馨小镇，是朋克都市，或是浮空岛屿。 每一处风景，都如梦幻中的秘境。 即使是火柴盒，也能众木成林，为一处奇观。

一无所有的我们，从这里开始 愿每个人都能从MC中发现自己所热爱的东西，或建筑、或红石、或绘画、或代码。 查阅资料，学习建筑，绘画，命令，我们都是成长中的玩家。

摒弃互联网的信息轰炸，让浮躁的心灵稍息。 在你的脑海深处，一定有数不尽的天马行空，在这里去实现和分享。

拉进网络的距离，在这里成为朋友、邻居。 少些戾气，多些合作，冷静处事，睦邻友好。

如果你对我们的理念深以为然，欢迎你的到来！ 为了实现这样的梦想，我们需要志同道合的伙伴：

我们需要有独特想法的玩家来一起策划和完善游戏内容。 我们需要会些文笔或视频的玩家来帮助宣传。 我们需要想学编程的玩家来一起用插件实现想法。

尽管现在一无所有， 我们来一起学习、讨论、设计属于自己的服务器， 享受成长的过程，这就是最棒的游戏体验。

可大致分三部分内容：

  1. 宣传：文案、视频、论坛贴吧网站招新。
  2. 策划：理念讨论、游戏机制、玩法设计、活动策划、城市设计规划。
  3. 技术：建筑学习、红石科技、命令方块、插件编程。
