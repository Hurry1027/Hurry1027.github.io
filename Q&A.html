# 会飞的史蒂夫

# 基岩版 | 生存休闲物语 | 死亡不掉落

# Q：作为服主，你为什么要开服务器？服务器的未来是什么？

A：我想建立一个更有人文关怀的MC服务器。

十年以来互联网高速发展，网络脱离了当初的懵懂纯真的状态，抽象亚文化大行于世，无论什么样的事物或想法都可能成为一场冷笑话 ，“典，孝，急，麻，绷”，充斥着对人与事物的不屑一顾与肤浅认知。这的不近人心的环境下，人也许可能会变的过于迎合别人，也可能会打不过就加入，选择加入这场抽象闹剧。焦虑的社区环境不知不觉让人心变的浮躁，简单的以物质价值来导引自己的劳动和实践，却不知生活意义何在。就如人为了掩饰内心的浮躁，会不断的追求新鲜刺激的感觉，这无可厚非，但这不会带来进步与发展。

所以，FlySteve的未来是什么呢？

我想要大家一起来建设，探索想象力，一起学习、进步与发展。

最为重要的是，我想让这里的人们能够友善交流，互相照顾，互相扶持。熟带生，佬带菜，最好能在任何地方都能见到热心的指路牌。简单的可以帮忙搬砖解压，进步的话可以一起研究建筑红石。每个人都可能出一份力，零零散散地将生产建设所需的工业体系建立起来，服务于我们各种因好奇心而起的建筑，实现我们的模拟与幻想。

Flysteve的玩法核心在于朴素的田园式建设，即小部分小部分的渐进式、成长式建设。换句话说就是，玩家一开始并没有建设大城市、大建筑的目标，而是什么时候有灵感了，就装饰性的建设一下。但是，随着时间流逝，慢慢的聚沙成塔，一个城市也有成型了，这即是我们的成长。无关技术力，无论建的多么奇怪都没关系。重点在于这些成果出自于我们自己之手，那都算我们最重要的成果。

无论何时，你对FlySteve印象深刻的，不是服务器有多少精妙的机器或宏伟的建筑，而是因为这里有一群友好且温暖的玩家，每个人都能在这里找到适合自己的角落安稳栖息。

# Q：我很无聊，不知道在服务器该玩什么

A：你可以参考宣传文中的玩法推荐，去尝试一些开创性的玩法， 尝试与突破是生命活力的来源。

你在生活中总会萌生一些奇奇怪怪的点子，灵感稍纵即逝， 敏锐的捕捉到这些新鲜的想法，那么MC便是你生活灵感的具现途径， 这就是自我价值的实践。

于我个人，我从来喜欢都是是自由度更高的游戏。比如很早之前的war3红警，我并非喜欢即时对战，而是因为这些游戏有较高的自由度，我能尽情的运用我的好奇心，实践不断冒出的新点子，好奇与创造是一件很有诱惑力的事情。

MC很小也很简单，它只有不到200MB，但它极为广阔，恰好成为你倾泻灵感的舞台。 于个人来说，上面说的或许是意义的一种解释。 而对于多人世界来说，人与人的联系极为复杂，MC的意义尚需我们向深处探寻。 或许是要我们领悟生而为人，认识世界，改造世界，劳动与实践的真谛。 建设家园，打怪升级，朋友联机冒险，这里的每个玩家，都在思考MC的意义，而这就是FlySteve存在的理由。

# Q: 为何主世界、下届、末地都有边界限制？

A：限制跑图，可以充分利用世界资源，压缩存档容量，优化存储空间便于维护，使服务器能长久的运行下去。（受益于此，三周目存档两年半至今，存档仅有1.8GB。而其他未做限制的服务器可能破10G甚至更大，甚至不得不重开新存档。） 挖矿和地图探索可以到资源世界，资源世界会不定期重置，请不要在资源世界留下重要物品。

# Q：为何限制末地城只能探索一次？

A：避免老玩家疯狂搜刮鞘翅，垄断鞘翅的行为。保证鞘翅的稀有度，同时也让后来的新人有不错的末地城探索体验。由于潜影贝可以复制了，如果你想要造潜影贝农场，你可以寻找一个未探索的没有末地船的末地城，联系服主给你进入的权限。或者找老玩家（门主不会建筑menzhu467等购买潜影贝。

# Q：为什么商店要限制柜台数量上限？

A：目前新店主限制10个柜台，看经营状况可额外申请20个，总计30。这是为了避免商店柜台的滥用。店主需要保证柜台经营的质量，认真打理经营店铺。如果经营态度很好，售卖的物品很有价值，可以建更多的店铺，再联系服主申请更多柜台。

柜台上限相当于制定一个恒定的世界规则。 这让个人在商店体系中有了能力上限和影响上限。也就是说限制了个人发展对服务器效率的影响程度。

一方面推动和鼓励玩家将个人发展的实力转为合作经营的实力。这期望让受限的个人效率转为寻求合作效率。鼓励联盟，部落经济体的发展。

另一方面也给更多人有机会参与到商店玩法，用更多的经营者填补市场空缺，而不是让头部几个实力特别强的人全覆盖到大部分商品，挤占大部分市场份额。

原版中鞘翅数量十分有限，官方商店出售是最效率的。但我们服并没有那么做。我们选择了给每个人尽量公平的空间，丰富多人体验。

这些都是理论看法，而且是保守性的，未来可能也会更改这些规则。

# Q：服务器为什么不大规模宣传招人？

A：一来不想有太多的人一起涌入，不好管理以及保证玩家素质，人多则乱是必然的。我期待是一小部分人形成稳定的交流环境后再有新人加入。二来在视频网站投稿是很好的宣传方式，但是我不会视频剪辑，也没有空余时间来学习做视频，所以就很佛系。如果有可能，很推荐你能把在服务器的游览经历做成视频并投稿宣传，而且我可以给一些的游戏内的奖励进行支持。这种记录游记的形式也是很棒的玩法。三是服务器目前性能不足，不足以带动太多人同时上线。四是因服务器理念，最好是先小而美然后再发展做大。
